# ⚽ Sistema de Predicción y Análisis de Fútbol 2026

Este proyecto es una herramienta integral de análisis deportivo que combina una base de datos robusta en **MySQL** con una interfaz interactiva en **Streamlit**.

## 👤 Autor
**Santiago Montoya** - Estudiante de Talento Tech.

## 🚀 Características
- **Base de Datos SQL:** Modelo Entidad-Relación con 4 tablas (Equipos, Partidos, Estadísticas, Cuotas).
- **Dashboard Interactivo:** Visualización en tiempo real de promedios, resultados y estadísticas.
- **Análisis Estadístico:** Uso de subconsultas eficientes para determinar el rendimiento de los equipos.
- **Control de Mercado:** Integración de cuotas de apuestas para análisis de valor.

## 🛠️ Tecnologías utilizadas
- **Backend:** Python 3.12+
- **Database:** MySQL Workbench
- **Frontend:** Streamlit
- **Librerías:** Pandas, Plotly, MySQL-Connector
